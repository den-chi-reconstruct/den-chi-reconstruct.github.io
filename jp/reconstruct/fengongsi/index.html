一般的な流れ

分公司の設立

  1. 設立場所を決定する（賃貸契約書の締結、或いは土地使用証明書の取得等） 関連部門：リース部門、国土局

  2. 総公司が分公司の設立に関する董事会決議

  3. 会社設立の審査 関連部門：総公司の所在地の対外経済貿易部門

  4. 会社設立登記 関連部門：分公司所在地の工商部門

  5. 他の登録手続 関連部門：分公司所在地の公安局、国税、地税、質量監督部門、銀行等

分公司の閉鎖抹消

  1. 総公司が分公司の閉鎖抹消に関する決定

  2. 分公司の主管税務局にて税務登記抹消を行う

  3. 分公司の取引銀行にて口座抹消を行う

  4. 分公司の主管工商部門、主管公安局、質量監督部門等にて登記抹消を行う

駐在員事務所の設立

  1. 設立場所を決定する（賃貸契約書の締結） 関連部門：リース部門

  2. 設立登記 関連部門：工商部門

  3. 他の登記手続 関連部門：公安局、銀行、国税、地税、税関、質量監督部門、外貨管理局等

駐在員事務所の閉鎖抹消

  1. 税務局にて税務登記抹消を行う

  2. 税関にて登記抹消を行う

  3. 銀行、外貨管理局等の部門にて登記抹消を行う

  4. 工商部門、公安局、質量監督部門等にて登記抹消を行う

主要な留意点

  • 分公司と法人会社との機能比較

    項目 法人会社 分公司
    法人資格 ある ない
    経営範囲 会社自体の要求に応じて申請可能 総公司の経営範囲を超えてはいけない
    契約書の締結 法人会社の名義で締結 総公司/分公司の名義で締結
    輸出入の経営権 ある（輸出入の通関を自社で申告可） ない（輸出入の通関を第三者への委託により代行実行が可能）
    貿易項目における外貨送金、入金 会社自体で外貨の送金と入金が可能（外貨入金を第三者への委託も可能） 不可（外貨の送入金を第三者への委託により代行実行が可能）
    発票の発行 可 可
    民事責任 独立で民事責任を負担する 不独立で民事責任を負担できない。総公司が自らの全ての資産を持って分公司の経営における債務に対して、責任を負担する

  • 駐在員事務所は法人資格がないため、利益を求める営業活動に従事してはならない。
